WeatherWear

An android app that takes data from the weather forecast and tells the user what to wear. The app uses temperature, precipitation, cloud cover and more to determine what the user should wear to prepare for the day's weather conditions. I created this app with a group of two other students and I primarily worked on the backend which is in Java. You can see our code on GitHub if you're curious.
